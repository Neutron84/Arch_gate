# Guide: Creating an Arch Linux Live USB

This guide shows you how to create a bootable USB drive from the official Arch Linux ISO.

### Prerequisites
- A USB drive (at least 4GB).
- A downloaded Arch Linux ISO file from the [official website](https://archlinux.org/download/).
- A tool to write the ISO to the USB drive.

### Recommended Tool: Balena Etcher
[Balena Etcher](https://www.balena.io/etcher/) is a free, open-source tool that works on Windows, macOS, and Linux. It is the safest and easiest option.

1.  **Download and install** Balena Etcher.
2.  **Open Etcher.**
3.  Click **"Flash from file"** and select your downloaded `archlinux-....iso` file.
4.  Click **"Select target"** and carefully choose your USB drive. **Double-check this step!**
5.  Click **"Flash!"** and wait for the process to complete.

Your bootable Arch Linux USB is now ready. You can use it to boot your computer and start the Arch Gate installation.````
